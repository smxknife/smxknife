## 狄克斯特拉算法(迪杰斯特拉算法)

### 关键点

* 广度优先解决最短路径问题，但是无法解决最优解（如：最短时间问题），这时候可以采用狄克斯特拉算法
* 关键理念：**找出图中最便宜的节点，并确保没有到该节点的更便宜的路径**
* 不能有**负权边**，负权边会导致路径断开

### 算法步骤

> 创建开销表

> 找到"最便宜"的节点，即可在最短时间内到达的节点

> 更新该节点邻居的开销

> 重复这个过程，直到对图中的每个节点都这样做

> 计算最终路径

### 术语

* 狄克斯特拉算法适用于每条边都关联权重的图
* 带权重的图称为加权图，不带权重的称为非加权图。（图可能存在循环）
* 狄克斯特拉算法只适用于**有向无环图**
